# Fuentes examinadas

Registro vivo de publicaciones, documentos y fondos ya revisados para la web del Alarde de San Marcial de Irun.

Este fichero no sustituye a las citas visibles en cada pagina. Su funcion es evitar duplicar trabajo, recordar que se extrajo de cada fuente y dejar constancia del criterio aplicado. Debe actualizarse cada vez que se procese una nueva publicacion, archivo o busqueda.

## Criterios

- `integrado`: la fuente ya se ha revisado y parte de su informacion se ha incorporado a la web.
- `registrado`: la fuente esta documentada para trazabilidad tecnica, permisos o control interno.
- `parcial`: se ha usado solo para datos concretos, descartando informacion no relevante.
- En programas de fiestas se registran solo datos utiles para el Alarde: mando, unidades, cantineras, musica propia, ritual, organizacion o cambios documentados.
- En fuentes fotograficas se registran permisos, procedencia, busqueda, filtros y criterios de atribucion.
- Si una fuente contiene informacion sobre Hondarribia u otros ambitos, solo se integra cuando se relacione directamente con el Alarde de Irun.
