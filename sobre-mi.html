Sobre este proyecto

Esta web nace como un espacio personal para reunir información, imágenes y recursos sobre el Alarde de San Marcial de Irun de forma ordenada y accesible.

El objetivo es mantener una página sencilla, clara y útil, con enlaces a fuentes externas cuando sea necesario contrastar datos, ampliar información o consultar materiales originales.

Este apartado puede completarse más adelante con una presentación personal, criterios del proyecto, contacto o cualquier otra información que quieras añadir.

Web informativa independiente. Contrasta los datos operativos con fuentes oficiales.
