Sobre este proyecto

Esta web nace como un espacio personal para reunir información, imágenes y recursos sobre el Alarde de San Marcial de Irun de forma ordenada y accesible.

El proyecto nace de una intención sencilla: reunir en un mismo lugar la información que suele estar dispersa en programas, prensa antigua, archivos fotográficos, publicaciones locales y recuerdos familiares. La web no pretende sustituir a esas fuentes, sino ayudar a encontrarlas, ordenarlas y leerlas con contexto.

A medida que se incorporen nuevos documentos, cada página irá creciendo con una redacción clara, citas verificables y materiales publicados solo cuando exista permiso o una licencia adecuada. Este apartado quedará abierto para completar más adelante la presentación personal, los criterios del proyecto y las vías de contacto.

Web informativa independiente. Contrasta los datos operativos con fuentes oficiales.
